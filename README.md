# PDFViewer Sample

This sample demonstrate how to implement the PDFViewer on a UWP app using MVVM pattern.

## Project structure
```
PDFViewerUWPDemo/
	PDFViewerUWPDemo/
		Assets/							- App resources
		Properties/						
		ViewModel/						- MVVM files
		images/							- icons
		App.xaml
		App.xaml.cs
		MainPage.xaml
		MainPage.xaml.cs
		PDFViewerUWPDemo.csproj
		Package.appxmanifest
	PDFViewerUWPDemo.sln
```
